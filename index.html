INTERIORAGENT: LLM Agent for Interior Design-Aware 3D Layout Generation

Generate realistic and functional indoor 3D scenes by iteratively writing, executing, and refining scene programs.

Abstract

Creating realistic and functional indoor scenes requires nuanced reasoning about spatial configurations and human-centered design principles. We present INTERIORAGENT, an LLM-agent framework for text-to-3D indoor scene generation that produces scenes with strong visual quality and functional utility.

Our approach combines a novel interior design-aware scene description language, design-aware synthesis tools, and optimization tools that improve ergonomics, realism, and prompt adherence. INTERIORAGENT also supports extensible tool integration, enabling new capabilities beyond standard layout generation.

Program-Based Generation

INTERIORAGENT iteratively synthesizes, executes, and refines scene code to generate 3D indoor layouts.

Design-Aware Tools

Interior design priors are encoded through a novel scene language and tools for retrieval, placement, and optimization.

Extensible Framework

The framework supports integration with LLMs, diffusion models, and 3D generation tools.
